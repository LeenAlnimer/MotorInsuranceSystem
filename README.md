# 🚗 Motor Insurance Management System

A full-stack ASP.NET Core MVC application for managing motor insurance operations.

---

## 📌 Features

- Manage Cars (Add / Edit / Delete)
- Manage Customers
- Manage Policies
- Dashboard with real-time statistics
- Entity relationships (Car ↔ Customer ↔ Policy)

---

## 🛠️ Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap

---

## 📊 Dashboard

- Total Cars
- Total Customers
- Total Policies
- Recent Policies
